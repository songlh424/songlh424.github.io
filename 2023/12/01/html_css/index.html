HTML

标签：

  1. 标题，<h1> ~ <h6>

  2. 段落，<p>

  3. 注释，<!-- xxx -->

  4. 换行，单标记，<br>

  5. 水平线，单标记，<hr>

  6. 网页标题，只能一个，<title>

  7. 元数据标签，放在head中，<meta>、<link>、<style>等

  8. 图片，单标签，<img>，<img src="xxxx", alt="" title="" width="" height=""/>

    • src，路径
    • alt，不能正常显示时给的提示
    • title，鼠标悬停时给的提示

  9. 文本格式化：<b>字体加粗，<i>倾斜，<u>下划线，<s>、<del>删除线，<me>、<strong>加重，<sub>上标，<sup>下标

  10. 链接，<a href="xxx"> xxx </a>

  11. 无序列表，<ul> <li> xxxx </li> <li> xxx </li></ul>

  12. 有序列表，<ol> <li> xxxx </li> <li> xxx </li></ol>

  13. 自定义列表，<dl><dt>xxx</dt><dd>xxx</dd></dl>，dt主题，dd描述

  14. 表格，<table><tr><td>单元格内容</td>...</tr>...</table>

    • <caption>表格标题
    • <th>特殊的单元格，标题，会水平居中且加粗
    • 表格划分，<thead>、<tbody>、<tfoot>
    • table属性：border边框像素大小，bordercolor边框颜色，align表格对齐，width、height表大小，bgcolor背景颜色，background背景图片，cellpadding单元格内边距，cellspacing单元格间距
    • tr属性：height，align，valign垂直方向对齐，bgcolor，background
    • td属性：colspan跨列，rowspan跨行

  15. 表单，

    • 下拉列表：<select> <option value="">内容</option> </select>
    • 多行文本：<textarea name="" cols="" rows="">内容</textarea>
    • 标签，用于关联文本和选项框：<input type="radio" id="man"><label for="man">男</label>
    • 隐藏域：<input type="hidden" name="country" value="Norway">
    • some属性，selected，checked，disabled，readonly，placeholder（描述输入域期待的值）

  16. 布局标签，<div>分块，<span>行内分区

  17. html5新增语义化布局标签，<header>、<nav>、<article>、<section>、<aside>、<footer>

  18. 视频和音频，<video> src="" controls>提示</video>

    • src，视频的URL
    • autoplay，视频就绪后马上播放
    • controls，显示控件
    • loop，播放完后再次播放
    • muted，视频音频静音
    • preload，页面加载时进行加载，并预备播放
    • poster，视频下载时显示的图像，直到用于点击播放按钮

    <audio src="">提示</audio>，属性同视频。

CSS

内容与样式分离，内容由html完成，样式由css完成。

  1. css样式表特征，继承性、层叠行、优先级

  2. 语法：选择器以及一条或多条声明，每条声明由一个属性和一个值组成。

  3. 引入方式

    1. 内联样式，通过html的style属性，属性值包含css属性。

    2. 内部样式，通过html的style标签引入，属性键值对放在双标签之间，放在head中。

    3. 外部样式，通过html的link标签引入，<link rel="stylesheet" href="xxx.css">，放在head中。

      优先级：内联样式 > 内部样式、外部样式；内部样式和外部样式后写的生效。

    4. 导入式，在css文件中导入css，@import "xxx.css"

      link和@import的区别：

      1. 加载顺序不同。link同时加载html和css文件，import先加载所有html再加载css
      2. 兼容性问题，link没有兼容性，import IE5以下支持
      3. link还可以引入其他内容，比如引入图标<link rel="icon" href="xx.icon">，import只能引入css
      4. js只能操作link引入的样式

  4. css样式、css属性，width、height、font-size、color、background-color、float:left（水平布局）、

  5. 选择器——选中写样式的元素

    1. 基础选择器：

      • 全局选择器 —— *{}，选中页面所有元素
      • 元素选择器 —— html所有元素都可以。例：div{}、p{}等
      • 类选择器 —— 利用html的class属性，除br外。.className{}
      • ID选择器 —— 利用html的id属性，除br外。#idName{}

      所列选择器的优先级是从低到高的

    2. 合并选择器 —— .className1, .className2 {共同样式}

    3. 关系选择器：

      • 子代选择器 >
      • 后代选择器 空格
      • 相邻兄弟选择器 +
      • 通用兄弟选择器 ~

    4. 伪类选择器

      同一标签，根据其不同状态，有不同的样式。这种叫“伪类”，用冒号表示。

      :link “链接”，超链接点击之前

      :visited “访问过的”，超链接被访问过后

      :hover “悬停”，鼠标放到标签上的时候

      :active “激活”，鼠标点击标签，但是不松手时

      CSS3新增伪类选择器：

      • first-child第一个子元素，last-child最后一个子元素，nth-child()第几个子元素
      • first-of-type、last-of-type、nth-of-type()——区分元素类型
      • foucus获取焦点，只适用于表单元素

  6. 基础属性

    • 字体font：font-size、font-weight、font-style（italic倾斜）、font-family、color（rgba(255, 0, 0, .66)设置颜色和透明度）
    • 文本属性：
      • text-align：center、left、right
      • text-decoration：underline（下划线）、overline（上划线）、linethrough（删除线）、none（不显示）
      • text-transform：uppercase（大写）、lowercase（小写）、capitalize（首字母大写）
      • text-indent:30px，首行缩进
    • 背景属性：
      • background-color
      • background-image : url(“./xxx.jpg/“)
      • background-repeat : no-repeat（不平铺）
      • background-size : x y
      • background-position : x y
      • background-attachment : fixed（背景固定）
    • 列表属性：
      • list-style-type：none（不显示）
      • list-style-image
      • list-style-position
    • overflow：指定内容溢出的操作。visible默认值可见，hidden不可见，scroll显示滚动条，auto自动
    • line-height：行高。
    • letter-spacing：字符间距
    • outline：设置input的外边框，none去掉边框
    • border-collapse：边框折叠，border-spacing：设边框之间距离
    • cursor：鼠标样式，pointer小手样式。

  7. 盒子模型

    所有html元素可以看作盒子。包括：边距（margin），边框（border），填充（padding）和实际内容（content）。

    1. 标准盒模型——w3c盒模型（浏览器默认）
      • content：width，height，min-width，max-width，min-height，max-height
      • border：
        • border-style：边框样式，必须属性，如solid实线，dotted点线。
        • border-color
        • border-width
      • margin：margin-top、margin-bottom、margin-left、margin-right。marin：0 auto水平居中。可取百分比，值是相对于父元素。
      • padding：类似margin。

  8. 浮动

    float定义元素在哪个方向浮动。只有左右两个方向。

    clear用来清除浮动的影响。

  9. 元素定位

    position属性用来指定一个元素在网页上的位置，一共5种定位方式

    • static，默认值。
    • relative，相对于自己原位置定位，原位置保留。
    • fixed，固定定位，相对于浏览器窗口定位。
    • absolute，相对于static定位以外的第一个父元素进行定位。父元素没有定位逐级向上找。
    • sticky，粘性定位，相当于相对定位（参照点为自己原来位置）和固定定位（参照点为浏览器窗口）的结合。
